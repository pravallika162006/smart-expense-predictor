# 💰 AI-Powered Smart Expense Predictor

An AI-powered financial analytics platform built using the MERN stack and Machine Learning for tracking expenses, predicting future spending, analyzing financial health, and generating intelligent budgeting insights.

---

# 🚀 Features

## 🔐 Authentication

* User Registration
* User Login
* JWT Authentication
* Password Encryption using bcrypt

---

## 💵 Expense Management

* Add Income
* Add Expenses
* Delete Transactions
* Category-Based Tracking
* Monthly Expense Filtering

---

## 📊 Analytics Dashboard

* Income vs Expense Analysis
* Pie Chart for Expense Categories
* Bar Graph for Financial Comparison
* Forecast Trend Graph
* Responsive Dashboard UI

---

## 🤖 AI / ML Features

* Expense Prediction using Linear Regression
* Financial Health Score
* Overspending Risk Detection
* AI Saving Suggestions
* Future Expense Forecasting

---

## 📁 Export Features

* Export Transactions as CSV
* Export Reports as PDF

# 🧠 Machine Learning Workflow

1. Expense history is collected from the user.
2. Data is sent to the Flask ML API.
3. Linear Regression model analyzes spending patterns.
4. Future expense predictions are generated.
5. Predictions are visualized in the dashboard.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/pravallika162006/smart-expense-predictor.git
```

---

## 2️⃣ Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## 3️⃣ Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## 4️⃣ ML Model Setup

```bash
cd ml-model
pip install flask flask-cors pandas scikit-learn
python app.py
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `server` folder.

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
